## 2025-05-15 - [CLI Summary Pattern]
**Learning:** For scientific CLI tools, presenting analysis results as a raw dictionary is overwhelming and less readable. A formatted list with explicit units (e.g., "s", "Hz", "dB") significantly improves the user's ability to quickly parse the output.
**Action:** Always format CLI summary outputs into a clean, labeled list with appropriate units and rounding. Use subtle visual cues like emojis to indicate process completion.
